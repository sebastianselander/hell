# Hell

A shell (again...) written in Haskell.

Thanks to the superior language it's the better version of [kjell](https://github.com/sebastianselander/kjell).

# Implemented
- `;`-list
- `&&`-list
- `||`-list
- Pipes
- Builtins (cd, exit, pwd)
- External commands
- Negation (`!`)
